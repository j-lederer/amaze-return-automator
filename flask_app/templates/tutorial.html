Step-by-Step Guide

Initial Setup

  1. 1 Go to Account and press the button to connect your Amazon account
  2. 2 Press on the button to Subscribe to Amaze-Software
  3. 3 Optional: Go to home and press 'Refresh Returns and Inventory' to get started immediately

Daily Usage

  1. 1 Scan the barcode of the return label - returns info will appear
  2. 2 If you wish to add the returned item to inventory, press 'Add To Queue'
  3. 3 When done with a batch of returns, press 'Add To Jobs'

Automated Processing

  1. 1 Inventory will be increased automatically at 12:00 AM EST
  2. 2 Returns data refreshes automatically every night
  3. 3 You can add multiple queues to the Jobs table as needed

Important Guidelines

Critical Timing: DO NOT edit your inventory between 11:00 PM - 2:00 AM EST on nights when you have jobs scheduled to execute at midnight.
Manual Execution Warning: If you manually execute tasks, DO NOT edit inventory in Amazon Seller Central within 1 hour before or 2 hours after execution.
Recommended: Allow the website to execute inventory increases automatically at midnight EST to avoid timing conflicts.

Data & System Information

Returns Data

• Accesses returns data from the last 60 days only

• Refresh will show FATAL status if no returns exist in last 60 days

• Data automatically refreshes every night at midnight

Inventory Requirements

• SKUs must be actively listed in your inventory

• Amazon Token must be valid (refresh annually)

• Failed updates typically due to unlisted SKUs

Troubleshooting

Inventory Update Fails

Ensure rejected SKUs are actively listed in inventory
Check if Amazon Token has expired (refresh in account)

Returns Refresh Fails

Verify you have returns in the last 60 days
Ensure Amazon Token is not expired
