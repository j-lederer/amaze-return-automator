Tutorial

Learn how to use the Warehouse Return Processing System

Step-by-Step Guide

1

Searching for Returns

Scan the barcode or enter the tracking ID in the search field. The system will automatically fetch all return information.

Tip: Press Enter after typing the tracking ID to search quickly.

2

Reviewing Return Details

Once found, review all the return information including address, item details, quantities, and refund amount.

Tip: Check if the item is already in the queue before adding it again.

3

Adding to Processing Queue

Click "Add to Queue" to include the item in the processing queue for automatic restocking.

Tip: Items in the queue will be processed at 12:00 AM EST daily.

4

Managing the Queue

Use "Add Queue to Jobs" to start processing, or "Clear Queue" to remove all items. Individual items can be removed using the trash icon.

Tip: Always review the queue before adding it to jobs.

Additional Features

Refresh Data

Manually refresh the system data to get the latest information from the warehouse management system.

Download Reports

Export queue data as PDF reports (full or slim versions) for record keeping and analysis.

Clear Search

Reset the search form and return information display to start fresh with a new search.

Queue Management

View all queued items, remove individual entries, or clear the entire queue. Monitor queue status in real-time.

Performance Tips

  • • Process returns in batches for better efficiency
  • • Use keyboard shortcuts (Enter to search)
  • • Regularly clear completed items from the queue
  • • Monitor refresh status for system health
